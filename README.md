# neubite

Portfolio site for http://www.bitemyapp.com/

## Prerequisites

You will need [Leiningen][1] 2.0 or above installed.

[1]: https://github.com/technomancy/leiningen

## Development

To start development:

```
lein repl
(use 'neubite.handler)
(boot)
```

## Running

To start a web server for the application, run:

    lein ring server
